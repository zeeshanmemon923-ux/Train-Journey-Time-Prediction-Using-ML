# Train Journey Time Prediction System

This is a Machine Learning-based desktop application that predicts train
journey duration based on:

-   Total Distance (km)
-   Total Number of Stops

The project uses Linear Regression and includes a GUI built with
Tkinter.

------------------------------------------------------------------------

## How to Run the Project

1.  Install required libraries:

    pip install -r requirements.txt

2.  Train the model (run once):

    python train_model.py

3.  Run the GUI:

    python gui_app.py

------------------------------------------------------------------------

## Project Files

-   Dataset1.csv → Dataset\
-   train_model.py → Trains and saves model\
-   model.pkl → Saved trained model\
-   gui_app.py → GUI application\
-   train_image.png → Image for welcome screen
